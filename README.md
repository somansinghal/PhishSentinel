# 🛡️ PhishSentinel

### AI Powered Cybersecurity Threat Detection Platform

PhishSentinel is a **modern cybersecurity dashboard** designed to detect phishing URLs, visualize cyber attacks, and analyze security threats.

It combines **phishing detection, threat intelligence visualization, and security analytics** into a single SaaS-style platform.

# ✨ Features

🔎 **Phishing URL Scanner**

Detect suspicious URLs using heuristic analysis.

🌍 **Live Cyber Attack Map**

Visualize simulated global cyber attacks.

📊 **Security Analytics Dashboard**

Track phishing and malware activity.

🖥 **Hacker Terminal Mode**

Interactive command-style interface.

⚡ **Live Cyber Threat Feed**

Real-time simulated attack notifications.

🎨 **Modern SaaS UI**

Glassmorphism cybersecurity dashboard.
